## Функционал
- Сортировка по выбору
- Фасеточная фильтрация по количеству пересадок (лишние фильтры с авиакомпаниями пропадают)
- Подгрузка новых перелетов

## Технологии
- React 18
- Redux-toolkit
- Vite
